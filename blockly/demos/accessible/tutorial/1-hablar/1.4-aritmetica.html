Tutorial 1.4: Aritmética


¡Bienvenido al tutorial 1.4 de Accessible Blockly!

Esta es una nueva ventana del navegador, si quieres cambiar de ventana puedes presionar Alt + Tab, y si quieres cerrar esta ventana puedes presionar Control + W.

Recuerda que los pasos deben realizarse en la ventana anterior llamada "Tutorial Blockly", por lo que te recomendamos que tengas ambas ventanas abiertas para que puedas seguir las instrucciones.

En el tutorial anterior aprendimos a crear un programa que valida si un número es par o impar. Para este tutorial aprenderemos a realizar operaciones aritméticas con números.

Esta vez encontrarás solo 2 bloques, uno de texto como en el primer tutorial y otro que ha encapsulado todos los bloques de texto dentro de un bloque de texto vacío del cual podrás encontrar todos los bloques de texto que ya conoces y muchos otros más.

Para realizar este tutorial, como siempre debemos seguir los siguientes pasos:

  1. Debemos empezar creando un nuevo grupo de bloques y esta vez seleccionamos el único bloque "decir EN BLANCO".
  2. Luego necesitamos agregar un bloque en la parte que dice "EN BLANCO", para esto debemos navegar dentro del bloque hasta encontrar la frase "Se necesita un valor cualquiera: Agregar bloque botón" y presionar Enter para ingresar a una lista con diferentes bloques que podemos agregar. En este caso seleccionaremos el bloque "1 + 1".
  3. Al igual que en el tutorial anterior, debemos modificar los números y la operación que queremos realizar. En este caso, cambiaremos "1 + 1" por "123 x 456".
  4. Finalmente ejecutamos el código presionando Control + Espacio y podremos escuchar el resultado de la operación.
  5. Recuerda que puedes explorar las distintas operaciones y probar con diferentes números. Además de poder escuchar el código generado en Python y JavaScript con las teclas Control + F1 y Control + F2 respectivamente.
  6. Este es el último tutorial de la sección "Hablar", ya que la siguiente sección es una tarea para desafiarte y que debes realizar tú solo. Si quieres volver a los tutoriales, puedes hacerlo desde la ventana "Selección de tutoriales". Pero antes no te olvides de cerrar esta ventana presionando Control + W.
