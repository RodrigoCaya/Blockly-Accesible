Tutorial 3.1: Asignar Texto


¡Bienvenido al tutorial 3.1 de Accessible Blockly!

Esta es una nueva ventana del navegador, si quieres cambiar de ventana puedes presionar Alt + Tab, y si quieres cerrar esta ventana puedes presionar Control + W.

Recuerda que los pasos deben realizarse en la ventana anterior llamada "Tutorial Blockly", por lo que te recomendamos que tengas ambas ventanas abiertas para que puedas seguir las instrucciones.

Bienvenido a la tercera parte de los tutoriales de Blockly Accesible. En esta parte aprenderemos a crear y utilizar variables, los cuales son bloques que nos permiten guardar información y son muy utilizados por todos los lenguajes de programación.

En este tutorial aprenderemos a crear variables, las cuales nos permiten guardar frases o palabras. Para esto debemos seguir los siguientes pasos:

  1. A partir de ahora, existe un último botón llamado "último botón: Agregar variable", que es donde crearemos nuestras variables. Para crear una variable debemos presionar este botón y agregarle el nombre de "saludo", luego presionamos "AGREGAR". Cabe destacar que la variable puede tener cualquier nombre, pero te recomiendo que sea un nombre que tenga sentido con la información que guardará.
  2. Ahora que creamos la variable llamada "saludo", podemos interactuar con ella. Para esto debemos crear un nuevo grupo de bloques y seleccionar el bloque de la categoría Variables "establecer saludo a EN BLANCO".
  3. Dentro encontraremos una lista desplegable con 3 valores: nuestra variable "saludo", la opción para cambiar de nombre la variable y la opción para borrar nuestra variable "saludo". Seleccionamos la primera opción: "saludo".
  4. Luego en la parte en blanco, agregamos el bloque de texto " " y lo cambiamos por "Hola". Con esto le estamos diciendo al computador que la variable "saludo" ahora tiene el valor de "hola".
  5. Por último, queremos escuchar el valor de nuestra variable. Para esto creamos un nuevo grupo de bloques y seleccionamos el bloque de texto "decir EN BLANCO". Luego agregamos el bloque de Variables con el nombre de nuestra variable, osea "saludo". Quedando el bloque como "decir saludo".
  6. Finalmente ejecutamos el código para escuchar el valor de nuestra variable, el cual es "hola". También recuerda que también puedes escuchar como sería nuestro programa en Python y JavaScript.
  7. Ahora ya sabes como almacenar un texto dentro de una variable y escuchar su contenido, pero esto no es lo único que puedes hacer con las variables, así que te invitamos a seguir aprendiendo en el siguiente tutorial. Pero antes no te olvides de cerrar esta ventana presionando Control + W.
